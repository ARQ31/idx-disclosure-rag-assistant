# 📈 IDX Disclosure RAG Assistant

A modern, fast, and intelligent Retrieval-Augmented Generation (RAG) application built with Streamlit. This tool helps investors, analysts, and the general public to quickly extract, analyze, and summarize critical information from Indonesian Stock Exchange (IDX) corporate disclosure PDFs (Keterbukaan Informasi).

## ✨ Features

- **Intelligent Financial PDF Parsing**: Utilizes `pdfplumber` to accurately preserve the structure of complex financial tables (assets, liabilities, EPS, etc.) preventing data corruption commonly found in standard OCR or PDF text extractors.
- **Advanced Text Normalization**: Custom Regex pipelines designed specifically for Indonesian financial terminology to cleanly format numeric data and keywords before embedding.
- **Multilingual Semantic Search**: Uses `paraphrase-multilingual-MiniLM-L12-v2` for highly accurate Indonesian language semantic embeddings and context retrieval.
- **Lightning Fast LLM Inference**: Powered by the Groq API, offering multiple LLM choices including Llama 3.1 (8B), Llama 3.3 (70B), and GPT OSS models.
- **Interactive UI**: A stunning, responsive "Glassmorphism" dark-themed UI built natively in Streamlit with intelligent "Quick Question" pills for seamless user experience.
- **Accurate Sourcing**: Prevents hallucinations by strictly grounding the LLM to the uploaded document and providing precise page number citations for every generated claim.

## 🛠️ Technology Stack

- **Frontend**: Streamlit, Custom CSS (Glassmorphism UI)
- **PDF Extraction**: `pdfplumber`, `pypdf`
- **Embeddings**: `sentence-transformers` (MiniLM)
- **Vector Search**: `scikit-learn` (Cosine Similarity)
- **LLM**: Groq Cloud API

## 🚀 How to Run Locally

### 1. Clone the repository
```bash
git clone https://github.com/ARQ31/idx-disclosure-rag-assistant.git
cd idx-disclosure-rag-assistant
```

### 2. Install dependencies
```bash
pip install -r requirements.txt
```

### 3. Setup API Keys
Create a `.streamlit/secrets.toml` file in the root directory and add your Groq API Key:
```toml
GROQ_API_KEY = "your-groq-api-key-here"
```

### 4. Run the app
```bash
streamlit run app.py
```

## 📸 Usage Guide

1. **Select Model**: Choose your preferred AI model from the sidebar. (Llama 3.1 8B is recommended for speed).
2. **Upload Document**: Upload any IDX corporate disclosure PDF (e.g., Buyback plans, Financial Reports, Corporate Actions).
3. **Ask Questions**: Click on one of the Quick Question pills or type a custom query. The AI will analyze the document and provide an answer backed by direct page citations.
